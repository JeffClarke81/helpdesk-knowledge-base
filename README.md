# Help Desk Knowledge Base

This repository contains step-by-step troubleshooting guides for common Windows and networking issues. The goal of this project is to demonstrate practical IT support skills used in real help desk environments.

## Project Purpose

As a recent graduate with a background in software development, I created this project to practice and document common IT support procedures. These guides simulate issues typically handled by help desk technicians and show my ability to troubleshoot, document solutions, and communicate technical steps clearly.

## Skills Demonstrated

- Windows troubleshooting
- Network connectivity diagnostics
- Command-line utilities
- User account management
- Technical documentation
- Problem-solving workflows

## Troubleshooting Guides Included

- **Fix No Internet Connection**
- **Map a Network Drive**
- **Flush DNS Cache**
- **Reset Local User Password**
- **Troubleshoot High CPU Usage**

Each guide includes:
- Problem description
- Possible causes
- Step-by-step resolution
- Verification steps
- Escalation procedures

## Technologies and Tools

- Windows 10 / Windows 11
