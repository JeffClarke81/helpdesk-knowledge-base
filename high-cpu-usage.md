# Troubleshoot High CPU Usage on Windows

## problem
The computer is running slowly, programs are lagging, or the system fan is running constantly due to high CPU usage.

----------

## Possible Causes
- Background applications consuming CPU
- Malware or unwanted software
- Outdated drivers
- Too many startup programs
- Windows updates are running

----------

## Step-by-Step Resolution

### 1. Open Task Manager
1. Press **CTRL + Shift + Esc**
2. Click **More Details** if the simplified view appears
3. Select the **Processes** tab

**Expected Result:** List of running processes appears with CPU usage percentages.

----------

### 2. Identify High CPU Processes
1. Click the **CPU** column to sort by usage
2. Look for processes using a large percentage of CPU

**Expected Result:** The application consuming the most CPU appears at the top.

----------

### 3. End Problematic Tasks
1. Select the process using excessive CPU
2. Click **End Task**

AVOID ENDING CRITICAL WINDOWS PROCESSES.

**Expected Result:** CPU usage decreases.

----------

### 4. Restart the Computer
Restarting clears temporary processes and resets system resources.

**Expected Result:** CPU usage returns to normal levels after reboot.

----------

### 5. Disable Unnecessary Startup Programs
1. Open **Task Manager**
2. Click the **Startup** tab
3. Right-click on unnecessary applications
4. Select **Disable**

**Expected Result:** Fewer programs start when the computer boots.

----------

### 6. Scan for Malware
Run a system scan using Windows Security.

1. Open **Windows Security**
2. Select **Virus & Threat Protection**
3. Click **Quick Scan**

**Expected Result:** Systems confirms no threats or removes detected malware.

----------

## Verification
- CPU usage in Task Manager remains below normal thresholds (typically under 20% when idle)
- Systems performance improves
- Applications open and run smoothly

----------

## Common Issues & Fixes

**CPU Usage remains High**
- Check for pending Windows updates
- Update device drivers
- Identify background applications consuming resources

**Unknown Process Using CPU**
- Research process name online
- Run an antivirus scan

**System Still Slow**
- Check RAM usage
- Check disk utilization

----------

## Escalation Path
If the issue scan

- Perform a full malware scan
- Update Windows and drivers
- Check hardware performance
- Escalate to system administrator or IT support WSpecialist

----------

## Notes
- Applies to Windows 10 and Windows 11
- Some processes are essential to system operation and should not be terminated
